# Splooosh

You play as an orange floating fish that terrorizes the world. What more could you want in a game?

Live: [Download](https://drive.google.com/open?id=1iEExPSas4td9tZZaRAXCnb_01Il5pYn4)
Github: [Source Code](https://github.com/Jonathannsegal/Sploosh)

![Banner image](/static/images/splooosh.gif)

Made with Unity3D and the Mapbox API, this was a fun weekend project for HackUIowa 2019. This game won best in class for Water Theme. If you want to play the game for yourself download and run it, or look at the source code if your interested in Mapbox's Unity3D integration.
